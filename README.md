# Lovdata

[Agent skill](https://agentskills.io) for looking up Norwegian laws using
Lovdata's public dataset. Works with any coding agent that supports the Agent
Skills standard.

Clone into your agent's skills directory and ask about Norwegian laws. See
[SKILL.md](SKILL.md) for available scripts and common law references.
